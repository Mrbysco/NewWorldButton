# Fast New World Button

## About ##

Adds a 'Create New World' button DIRECTLY to the pause menu, allowing you to abandon your world and quickly create a new one if you die in Hardcore mode! A great little utility mod for those who struggle with skill issues, or who just want to immediately generate new terrain for testing purposes!
